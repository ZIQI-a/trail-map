# 技能栈写法

## 前端技能

### React

普通写法：

```text
熟悉 React 组件开发，了解 Hooks 使用。
```

优化写法：

```text
熟悉 React 函数组件和 Hooks 开发，能按业务职责拆分页面、组件、hooks 和工具函数，处理复杂页面中的多组件状态联动。
```

项目支撑点：地图工作台拆分为 Header、地图、景点列表、详情面板、行程池、路线抽屉和完整行程配置等组件。

### TypeScript

普通写法：

```text
熟悉 TypeScript 基础类型。
```

优化写法：

```text
熟悉 TypeScript 在前端项目中的落地，能为接口响应、业务组件 Props、地图坐标、路线分段和行程节点建立清晰类型约束。
```

项目支撑点：`types/mapWorkbench.ts` 中维护城市、景点、路线、行程、用户态等类型。

### TanStack Query

普通写法：

```text
使用 TanStack Query 请求接口。
```

优化写法：

```text
熟悉 TanStack Query 的查询、变更、条件请求和缓存管理，能使用 queryKey、enabled、placeholderData 控制复杂页面的数据请求和切换体验。
```

项目支撑点：城市、景点、标签、POI 候选、收藏状态、打卡状态、用户行程和公开分享都通过 hooks 集中管理。

### 地图 SDK

普通写法：

```text
使用过百度地图 API。
```

优化写法：

```text
有百度地图 JavaScript API 接入经验，能处理 SDK 动态加载、地图初始化、Marker、Polyline、坐标系转换、地图选点和路线可视化。
```

项目支撑点：`lib/baiduMap.ts` 负责 SDK 加载和坐标点创建，`BaiduMapStage` 负责地图展示和路线绘制。

### Ant Design 与 CSS Modules

普通写法：

```text
熟悉 Ant Design 和 CSS Modules。
```

优化写法：

```text
能结合 Ant Design 和 CSS Modules 开发业务页面，使用组件库承接表单、弹窗、反馈状态，用模块化样式处理地图浮层、复杂布局和响应式适配。
```

项目支撑点：地图工作台、个人中心、收藏/打卡、我的行程、后台管理页面均使用该组合。

## 后端技能

### Spring Boot

普通写法：

```text
了解 Spring Boot 后端开发。
```

优化写法：

```text
了解 Spring Boot RESTful API 开发，能按 Controller、Service、Mapper、Entity 分层实现业务接口，并与前端完成接口联调。
```

项目支撑点：城市、景点、路线规划、用户认证、收藏打卡、行程管理和后台管理接口。

### MyBatis-Plus 与 MySQL

普通写法：

```text
了解 MyBatis-Plus 和 MySQL。
```

优化写法：

```text
了解 MyBatis-Plus 常规 CRUD 和条件查询，能根据业务关系设计 MySQL 表结构、外键、唯一约束和基础索引。
```

项目支撑点：city、spot、spot_tag、route_record、route_segment、app_user、user_trip、user_trip_item 等核心表。

### Flyway

普通写法：

```text
了解数据库迁移工具。
```

优化写法：

```text
了解 Flyway 数据库版本管理，能通过 V 版本迁移脚本维护表结构、种子数据和后续字段演进，避免不同环境手工改表不一致。
```

项目支撑点：项目已有 V1-V13 迁移脚本，覆盖核心表、用户域、路线持久化、公开分享和全量城市种子数据。

### Spring Security

普通写法：

```text
了解登录认证。
```

优化写法：

```text
了解基于 Bearer Token 的认证流程，能将自定义 Token 校验接入 Spring Security 上下文，并配合前端请求层完成登录态接口访问。
```

项目支撑点：后端 Bearer Token 过滤器校验用户，前端请求封装自动携带 Authorization 头。

## 简历技能栏推荐版本

前端岗位版本：

```text
- 熟悉 React、TypeScript、Vite 前端开发，能基于 Hooks 和组件拆分实现复杂业务页面。
- 熟悉 TanStack Query 服务端状态管理，能处理条件请求、缓存复用、变更操作和接口错误状态。

- 有百度地图 JavaScript API 接入经验，能处理 SDK 动态加载、坐标转换、Marker、Polyline、地图选点和路线可视化。

```

```text
- 了解 Flyway 数据库迁移和 Spring Security Token 认证，能完成登录态接口和权限控制的基础闭环。
- 熟悉前后端分离项目结构，能从接口契约、数据类型、错误处理和环境变量配置角度保障联调稳定性。
```
