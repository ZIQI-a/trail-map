## 术语表

| 术语 | 英文 | 说明 |
| --- | --- | --- |
| 行迹旅图 | TrailMap | 本项目名称，全国旅游地图规划应用 |
| 景点 | Spot | 旅游景点，项目核心数据实体 |
| 城市 | City | 旅游城市，景点的上级组织单位 |
| 标签 | Tag | 景点特征标签（如"适合拍照"、"免费景点"） |
| 行程 | Trip | 用户保存的一次完整旅行规划 |
| 行程池 | Trip Pool | 用户临时选择的景点集合，用于规划前的景点收集 |
| 路线规划 | Route Plan | 根据景点列表计算最优路线的过程 |
| 自由模式 | Free Mode | 用户自由选择景点顺序的路线规划模式 |
| 日程模式 | Schedule Mode | 按天自动安排景点的路线规划模式 |
| 打卡 | Check-in | 用户标记已到访某景点的行为 |
| 足迹 | Footprint | 用户打卡记录的总和，可在地图上可视化展示 |
| 收藏 | Favorite | 用户标记感兴趣的景点 |
| POI | Point of Interest | 兴趣点，地图上的具体位置 |
| GCJ-02 | - | 国测局坐标系，国内地图通用坐标系 |
| BD09 | - | 百度地图坐标系，在 GCJ-02 基础上加密 |
| WGS84 | - | GPS 全球定位系统使用的坐标系 |
| HMAC | Hash-based Message Authentication Code | 基于哈希的消息认证码，用于 Token 签名 |
| PBKDF2 | Password-Based Key Derivation Function 2 | 密码哈希算法 |
| Flyway | - | 数据库版本迁移工具 |
| MyBatis-Plus | - | MyBatis 的增强工具，简化 CRUD 操作 |

## 环境变量速查

### 后端环境变量

```bash
# 数据库配置
TRAILMAP_DB_HOST=localhost
TRAILMAP_DB_PORT=3306
TRAILMAP_DB_NAME=trailmap
TRAILMAP_DB_USERNAME=root
TRAILMAP_DB_PASSWORD=your_password

# 百度地图
BAIDU_MAP_SERVER_AK=your_baidu_map_server_ak

# 认证
TRAILMAP_AUTH_TOKEN_SECRET=your_secret_key
```

### 前端环境变量

```env
# API 地址
VITE_API_BASE_URL=http://localhost:8080

# 百度地图
VITE_BAIDU_MAP_AK=your_baidu_map_ak
```

## 常用命令速查

### 后端

```bash
# 安装依赖
mvn clean install -DskipTests

# 运行应用
mvn spring-boot:run

# 运行测试
mvn test

# 打包
mvn clean package -DskipTests

# 查看 Flyway 状态
mvn flyway:info
```

### 前端

```bash
# 安装依赖
npm install

# 开发服务器
npm run dev

# 代码检查
npm run lint

# 生产构建
npm run build

# 预览生产构建
npm run preview
```

### Docker

```bash
# 构建后端镜像
docker build -t trailmap-backend ./backend

# 构建前端镜像
docker build -t trailmap-frontend ./frontend

# 使用 Docker Compose
docker-compose up -d

# 查看日志
docker-compose logs -f backend
```

### 数据库

```bash
# 连接数据库
mysql -h localhost -P 3306 -u root -p trailmap

# 备份数据库
mysqldump -u root -p trailmap > backup.sql

# 恢复数据库
mysql -u root -p trailmap < backup.sql
```
