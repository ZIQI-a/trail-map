# 后端文档索引

本文件是后端 Harness 工程入口。AI 在处理 `backend/` 目录任务时，应先读取 `backend/AGENT.MD`，再按任务需要查阅本索引中的详细文档。

- 后端 AI 约束入口：`backend/AGENT.MD`
- 项目总需求：`require/全国旅游地图项目.md`
- 数据库草案：`require/数据库设计.md`
- 分阶段计划：`docs/开发规划.md`

## 详细规范

- 后端工程规范：`docs/backend/后端工程规范.md`
- 后端本地配置说明：`docs/backend/后端本地配置说明.md`
- 地图 API 接入说明：`docs/地图API接入说明.md`

## 当前阶段关注点

- 当前已进入阶段 3，后端已接入 MySQL、MyBatis-Plus 和 Flyway。
- 数据库初始化通过 Flyway 迁移脚本执行，不再推荐手动维护散落 SQL。
- 本地运行前需要先查看 `docs/backend/后端本地配置说明.md`。

## 查阅指引

- 做实体或表结构前：查看 `require/数据库设计.md` 和 `docs/backend/后端工程规范.md`。
- 做接口前：先确认接口属于当前开发阶段，再补充或查看接口契约文档。
- 做路线规划前：查看 `docs/地图API接入说明.md`。
- 修改依赖、端口、包结构前：同步检查 `docs/开发规划.md`。
